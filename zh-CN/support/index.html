赞助 immer

Immer 目前在 GitHub 上有 350000 个项目依赖于它，每月的下载量接近 10000000 次。然而，只有一开始两天的开发得到了赞助（由 Mendix），之后的所有开发和维护都是用爱发电。

如果您喜欢 Immer，并且对这个包心存感激，或者想确保它的使用寿命，请考虑在 https://opencollective.com/immer 上赞助或使用 PayPal 进行一次性捐赠。
